# Amour Melodie Records

World-class piano music label website with dark luxury design.

## Quick Links

- **Live Site:** https://amour-melodie-records.vercel.app
- **GitHub:** https://github.com/misto-guest/amour-melodie-records

## Design Features

- Dark luxury theme with gold accents
- Premium typography (Syne + Cormorant Garamond)
- Smooth animations and interactions
- Mobile responsive design
- Single HTML file (27KB)
- Loads in <1 second

## Deployment

This site is deployed on Vercel and connected to GitHub.
Every push to the main branch triggers an automatic deployment.

## Technology

- Pure HTML with embedded CSS/JavaScript
- Tailwind CSS (via CDN)
- Google Fonts
- No build process required
